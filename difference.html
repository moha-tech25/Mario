Le DEVELOPPEMENT WEB

Le développement web, c’est un processus de création de sites web et d’applications web qu’on utilise sur Internet.
Il y a 3 grandes parties dans le développement web :

  1. Front-End (ce que l’utilisateur voit)
    C’est la partie visible du site : le design, les textes, les images, les boutons… tout ce qu’on voit et avec quoi on interagit. Langages utilisés : HTML (structure des pages) CSS (design, couleurs, mise en forme) JavaScript (animations, interactions, boutons qui réagissent)
  2. Back-End (ce qui se passe en coulisses)
    C’est la partie invisible pour l’utilisateur. Elle gère la logique, les bases de données, les comptes utilisateurs, etc. Langages utilisés : PHP, Python, Node.js, Java, etc. Bases de données : MySQL, MongoDB, etc.
  3. Full-Stack (les deux à la fois)
    Un développeur Full-Stack s’occupe à la fois du Front-End et du Back-End.

La PROGRAMMATON MOBILE

La programmation mobile est l'ensemble des techniques et des langages utilisés pour créer des applications destinées à être utilisées sur des appareils mobiles tels que les smartphones et les tablettes.En résumé, Programmation: Écrire des instructions (du code) que l’ordinateur ou le téléphone peut comprendre. Mobile: Tout ce qui concerne les téléphones portables, tablettes, etc. Donc, la programmation mobile, c’est écrire du code pour créer des applications mobiles (comme WhatsApp, Facebook, TikTok, etc.). Deux types d'applications mobiles :

  1. Applications natives : Développées spécialement pour un système (Android ou iOS). Langages : Java ou Kotlin (Android), Swift ou Objective-C (iOS).
  2. Applications hybrides (ou cross-platform) : Une seule application qui fonctionne à la fois sur Android et iOS. Outils : React Native, Flutter, Ionic.

La différence entre le développement web et la programmation mobile

  1. Développement Web : Créer des sites web et applications web accessibles via un navigateur (Chrome, Firefox, etc.). Exemples : Sits internet (Google, Wikipédia). Applications web (Gmail, Facebook Web, YouTube Web). Technologies utilisées : HTML, CSS, JavaScript (pour l'interface). PHP, Python, Node.js (pour le côté serveur). Frameworks : React.js, Angular, Vue.js (Front-End), Laravel, Django (Back-End). Nécessite une connexion Internet. Accessible sur tous les appareils via un navigateur (mobile, ordinateur, tablette).;
  2. Programmation Mobile : Créer des applications qui s’installent sur les smartphones ou tablettes. Exemples : WhatsApp, TikTok, Instagram (installées sur téléphone). Applications bancaires. Technologies utilisées : Java, Kotlin (Android). Swift, Objective-C (iOS). Flutter, React Native (pour les deux plateformes à la fois). Téléchargement depuis Play Store ou App Store. Fonctionne parfois même sans Internet.

La différence principale entre le developpement web et la programmation mobile réside dans le fait que le developpeur crée des sites ou applications web accessible via un navigateur web et le programmeur quant a lui crée des applications mobiles.
